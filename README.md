# CompoundEffectCalculator

 Below is an example of a Java application that computes the compound effect (compound interest). It requires the user to input the principal amount, annual interest rate, the number of times that interest is compounded per year, and the total number of years. The application will then calculate and display the total amount.

 Features:

User Inputs:

Principal amount
Annual interest rate (in percentage)
Compounding frequency (number of times per year)
Total number of years
